How It Works

Three steps. Zero friction.

Get a comprehensive security report for your OpenClaw setup in under two minutes.

1

Install the plugin

One command pulls the plugin from GitHub into your OpenClaw environment.

openclaw plugins install github:XiaoYiWeio/openclaw-deepsafe
2

Run the scan

A single command kicks off the full preflight security analysis across all four dimensions.

openclaw deepsafe scan
3

Review the report

A beautiful, interactive HTML report opens automatically in your browser with actionable findings.

→ report opens in browser
Scan Dimensions

Four layers of defense.

Every scan covers deployment, skills, model behavior, and memory to give you complete situational awareness.

Deployment Config

Examines your gateway exposure, authentication settings, TLS configuration, and exposed ports for misconfigurations.

Gateway Exposure Auth Checks TLS Verification Port Scanning

Skill / MCP Analysis

Scans registered skills and MCP servers for prompt injection vectors, dangerous runtimes, and overly broad permissions.

Prompt Injection Dangerous Runtimes Permission Analysis

Model Safety

Runs four specialized benchmarks against the configured model to evaluate behavioral risks in production contexts.

Persuasion Sandbagging Deception Hallucination

Memory & History

Inspects conversation history and memory stores for leaked secrets, PII exposure, and injection persistence patterns.

Secret Leaks PII Detection Injection Persistence

Features

Built for developer workflow.

No cloud accounts, no config files, no waiting. Just the security insights you need.

Zero Cloud Dependency

Everything runs on your machine. No data leaves your environment, ever.

< 2 Min Scan Time

Full four-dimension analysis completes in under two minutes on typical setups.

Auto-Opens HTML Report

A rich, interactive report launches in your browser the moment the scan finishes.

Result Caching (TTL)

Benchmark results are cached with configurable TTL so repeat scans are near-instant.

Python 3 Only

No Docker, no Node, no extra dependencies. If you have Python 3, you're good to go.

Config Auto-Detection

Automatically discovers your OpenClaw configuration—gateway, skills, models, and memory stores.
